# ITSM Incident & SLA Performance Dashboard | Power BI

## Project Overview

This project is an **interactive 3-page Power BI dashboard** built to analyze **incident management, SLA compliance, team performance, and service quality metrics**.

The dashboard simulates a real-world **IT Service Management (ITSM)** reporting solution used by support managers and stakeholders for operational decision-making.

This project closely aligns with enterprise incident management workflows such as **ServiceNow ticket analysis, SLA breach monitoring, and support team performance reporting**.

---

## Business Objective

The objective of this dashboard is to provide insights into:

* total ticket volume
* open vs closed incidents
* SLA compliance
* average resolution time
* ticket aging
* reopened ticket trends
* team and agent performance
* region-wise workload distribution

## Dataset

A realistic ITSM-style dataset with **500 incident records** was created for analysis.

Fields include:

* Ticket ID
* Created Date
* Closed Date
* Priority
* Category
* Status
* Assigned Group
* Agent Name
* SLA Status
* Resolution Time
* Reopened Flag
* Region
